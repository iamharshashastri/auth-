Everything you need

AuthFlow handles the hard parts so you can ship faster.

🔐

Multi-Provider OAuth

Support Google, GitHub, Microsoft, Discord, LinkedIn, Patreon, Reddit, and X out of the box.

🏢

True Multi-Tenancy

Each app gets isolated user pools, separate API keys, and independent provider configs.

🔑

JWT-Based Sessions

Issued JWTs that your app can verify independently, with configurable expiry.

📡

Webhooks

Real-time event delivery for user lifecycle events with HMAC-signed payloads.

🛡️

bcrypt Passwords

App user passwords are bcrypt-hashed with per-app salts — never stored in plaintext.

⚡

Edge Functions

Globally distributed auth running on Supabase Edge — sub-50ms response times worldwide.
